Welcome to Whiskit Task Manager

Whishkit Task Manager is your all-in-one solution for efficient task management, seamless collaboration, and streamlined workflows. Whether you're handling daily tasks, managing projects, or approving transactions, Whishkit has you covered.

🚀 Why Choose Whishkit?

  • 🔐 User-Based Access: Each user gets a personalized experience with assigned tasks, contacts, and projects.
  • 📋 Bulk Task & Data Import: Add multiple tasks, contacts, guidelines, and transactions directly from a Google Sheet, making bulk entry effortless.
  • ✅ Smart Task Management: Mark tasks as done, edit details, and add comments—all directly within the app.
  • 📞 Contact Assignments: Users only see the contacts they are assigned to, ensuring privacy and focused task management.
  • 📂 Reusable Project Templates: Create project templates and use them anytime to quickly set up tasks.
  • 📜 Company Guidelines & SOPs: Store and access your company’s Standard Operating Procedures (SOPs) to maintain consistency.
  • 💰 One-Click Transaction Management: View all transactions in one place and approve them directly from the frontend.
  • 📝 Approved Transactions View: Get a separate section to track and manage all approved transactions easily.
  • 📱 Mobile-Friendly Interface: Manage and mark tasks as done effortlessly from your mobile device, making work convenient on the go.
